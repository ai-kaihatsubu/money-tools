# 013-discount-calculator（割引・セール価格計算機）

元値とX%OFF（または○割引）から割引後の価格・割引額を自動計算できる無料ツールです。

## 機能

1. ①割引後価格を計算: 元値を入力し、割引（%OFF）を1つ以上追加すると、上から順に重ね掛けされた
   割引後価格・割引額（合計）・各ステップの内訳を表示します。「税込価格も表示する」をオンにすると
   指定税率での税込価格も表示します。
2. ②割引率を逆算: 元値とセール価格から、割引率（%OFF）・割引額・「○割引」表記を逆算します。

検算例: 1000円の20%OFFは、割引額200円・割引後価格800円。

## プライバシー

- 入力した金額・割引率は外部送信・保存しません。すべて端末内（ブラウザ）で処理します。
- localStorageに保存するのはダーク/ライト表示設定のみ（入力金額は含まない）。

## 収益設計（プレースホルダ）

- AdSense: コンテンツ上下に2枠（`monetization.js`の`ADSENSE_CLIENT_ID`に本番IDを設定済み。広告ローダーは1回だけ注入）
- アフィリエイト: 「おすすめ」枠（ポイント還元・クーポンサービス等想定、`AFFILIATE_ITEMS`未設定）
- Stripe お布施: 任意のご支援（`STRIPE_DONATION_URL`未設定）

## 公開手順（社長作業）

1. GitHub新規リポジトリ `discount-calculator` を作成し、GitHub Pagesで公開
2. `index.html`等の`canonical`/OGP URLが実際の公開URLと一致するか確認
3. `operator.html`の運営者情報・特定商取引法に基づく表記を記入
4. AdSense審査申請（クライアントIDは設定済み）→ 通過後、広告配信を確認
5. AdSense配信開始に伴うCookie同意（CMP）対応の検討
6. アフィリエイト提携（ポイント還元・クーポンサービス等）→ `AFFILIATE_ITEMS`を実リンクに更新
7. Stripeお布施Link発行 → `STRIPE_DONATION_URL`設定
8. 公開前に開発用「Proフラグを切替（開発用）」ボタンの削除を検討
